Alert box

The alert box is as the name suggests an box that appears upon loading the webpage and provides the user with some text which is an alert.

The alert box can be applied to a webpage by using javascript.

The javascript for it can be implemented by using the following syntax :

window.alert("The alert")

Prompt box

The prompt box provides a prompt on the webpage that asks for an input from the user along with a message. The input can be in different form of data types.

It can also be implemented by using javascript.

In general, the javascript for it can be implemented by using the following javascript syntax :

prompt("The message for prompt")

Difference

The basic difference between an alert box and an prompt box is that an alert box is used if we want that the information comes through to the user. A prompt box is used when we want the user to input a value before they access the page. In an alert box only a message is delivered to the user while in an prompt box along with an message the user has to input a value.
